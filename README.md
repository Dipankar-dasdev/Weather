# Weather App (API Based)

This is a simple Weather App built using HTML, CSS, and JavaScript.  
It uses the OpenWeather API to fetch real-time weather data for any city entered by the user.

I built this project to practice working with APIs and improve my JavaScript skills.

------------------------------------------------------------

## Live Demo

You can view the live project here:

Live Link: https://dipankar-dasdev.github.io/Weather/

------------------------------------------------------------

## Features

- Enter city name
- Shows current temperature (in Celsius)
- Shows humidity
- Shows weather condition
- Displays weather icon
- Error handling for:
  - Invalid city name
  - Empty input
  - API errors
- Loading state while fetching data
- Responsive design

------------------------------------------------------------

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API

------------------------------------------------------------

## How to Run This Project Locally

1. Clone the repository.
2. Go to https://openweathermap.org/ and create a free account.
3. Generate your API key.
4. Open script.js.
5. Replace:

   const API_KEY = "YOUR_API_KEY_HERE";

   with your API key.
6. Open index.html in your browser.
